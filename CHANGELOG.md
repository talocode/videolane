# Changelog

## v0.1.0 (2026-07-17)

Initial release.

### Features

- Script-to-recording pipeline
- Browser recording with Playwright (dry-run + manual modes)
- ffmpeg video rendering/export
- Caption generation (SRT, VTT, JSON)
- Background audio generation
- Shorts cutdown planning and export
- Thumbnail brief generation
- YouTube metadata generation (title, description, tags, pinned comment)
- YouTube publishing package
- 5 project templates
- CLI with 16 commands
- TypeScript SDK
- HTTP API (port 3110)
- MCP server (12 tools)
- Python package (talocode-videolane)
- Deterministic demo mode
- Doctor command for environment checks
