# Roadmap

## v0.1.0 (current)

- Script-to-recording pipeline
- Browser recording with Playwright (dry-run + manual modes)
- ffmpeg video rendering
- Caption generation (SRT, VTT, JSON)
- Background audio generation
- Shorts cutdown planning
- Thumbnail brief generation
- YouTube metadata generation
- YouTube publishing package
- 5 project templates
- CLI, SDK, API, MCP
- Python package

## v0.2.0

- Full Playwright browser recording with video capture
- ffmpeg caption burn-in
- Multi-language captions
- Background music library
- Shorts auto-cut with ffmpeg
- YouTube Data API upload
- ScreenLane integration

## v0.3.0

- Transcript-based captions (Whisper)
- Thumbnail image generation
- Video templates library
- Multi-track audio mixing
- Advanced transitions
- Batch processing

## v1.0.0

- Full nonlinear editor capabilities
- Real-time preview
- Plugin system
- Collaborative editing
- Cloud rendering option
