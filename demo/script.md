# VideoLane Demo Script

## Introduction
VideoLane is an agentic video production engine for product demos, tutorials, and launch videos.

## Script Parsing
VideoLane parses markdown scripts and production packages into structured video plans.

## Browser Recording
Record browser actions using Playwright. Supports dry-run and manual modes.

## Captions
Generate SRT, VTT, or JSON captions from scripts or plans.

## Background Audio
Generate ambient or cinematic background audio locally.

## Shorts
Generate shorts cutdown plans and export vertical clips.

## Thumbnails
Generate thumbnail briefs with text options, colors, and layout.

## YouTube Metadata
Generate title options, descriptions, tags, and pinned comments.

## YouTube Package
Package everything for manual YouTube upload.

## End
