# VideoLane Demo Video

## Introduction
VideoLane is an open-source agentic video production engine for product demos, tutorials, and launch videos.

## Script Parsing
Paste a markdown script. VideoLane parses it into structured scenes with timestamps, narration, and browser actions.

## Browser Recording
Record browser actions using Playwright. Supports headless and manual login modes. Dry-run shows actions without launching a browser.

## Captions
Generate SRT, VTT, or JSON captions from scripts. No API key needed for script-based captions.

## Background Audio
Generate ambient or cinematic background audio locally using ffmpeg. No API keys required.

## Shorts
Generate shorts cutdown plans from your main video. Export vertical clips at 1080x1920.

## Thumbnails
Generate thumbnail briefs with text options, colors, layout, and Canva instructions. Includes SVG mockups.

## YouTube Metadata
Generate title options, descriptions with UTM links, tags, pinned comments, chapters, and upload checklists.

## YouTube Package
Package everything into a folder ready for manual YouTube upload.
